URL Endpoints

Request URL Method Params Request Body Data Format Response Body Data Format
Return a list of ALL movies to the user /movies GET None A JSON object holding data about all movies in the movies array
Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user /movies/:title GET None A JSON holding data about a single movie by title
Return data about a genre (description) by name/title (e.g., “Thriller”) /movies/genre/:genreName GET None A JSON holding data about the genre by name/title
Return data about a director (bio, birth year, death year) by name /movies/director/:directorName GET None A JSON holding data about the director

Allow users to update their user info (username) /users/:id PUT None A text message indicating their profile has beeen updated
Allow users to add a movie to their list of favorites (showing only a text that a movie has been added—more on this later) /users/:id/:movieTitle PUT None A text message indicating that a movie has been added to their Movies List
Allow users to remove a movie from their list of favorites (showing only a text that a movie has been removed—more on this later)> /users/:id/:movieTitle PUT None A text message indicating that a movie has been removed from their Movies List
Allow existing users to deregister (showing only a text that a user email has been removed—more on this later) /users/:id DELETE None A text message indicating that their profile has been removed
